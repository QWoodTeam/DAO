# Qwood DAO

DAO smart-contract for Qwood.  
With using [Truffle framework](http://truffleframework.com/).

## Install

```
$ npm i -g truffle
$ npm i
```

## Quick Usage

#### Run in testrpc 

```
$ npm run develop
> compile
> migrate --reset
> test
```
#### Run in Ropsten test network
```
$ npm run ropsten
> compile
> migrate --reset
```
